<TopBlock>

Prisma Studio 是数据库中数据的可视化编辑器。请注意，Prisma Studio 不是开源的，但是您仍可以在[`prisma/studio`](https://github.com/prisma/studio) 仓库中创建问题。

您可以通过两种方式运行它:

1. 在终端中运行 `$ npx prisma studio`。
2. 从安装程序中安装[桌面应用程序](https://github.com/prisma/studio/releases)。支持 Windows、MacOS 和 Linux。

</TopBlock>

## 故障排除

### 终端: 无法运行脚本 / Prisma Client 请求中出现错误

缓存问题可能会导致 Prisma Studio 使用较旧版本的查询引擎。您可能会看到以下错误:

```
Error in request:  PrismaClientKnownRequestError: Failed to validate the query Error occurred during query validation & transformation
```

要解决此问题，请删除以下文件夹:

- MacOS 和 Linux 上的 `~/.cache/prisma`
- Windows 上的 `%AppData%/Prisma/Studio`
